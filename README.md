# Weekly Planner (주간 일정관리 앱)
일주일 단위로 일정을 관리하고 메모를 작성할 수 있는 웹 애플리케이션
<br>

## 주요 기능
- 주간 달력 뷰
- 메모 작성 및 관리
- 드래그 앤 드롭으로 일정 이동
- 실시간 데이터 저장 (Firebase)
- 반응형 디자인

## 기술 스택
**Frontend**
- React 18
- Vite
- Tailwind CSS (or CSS Modules)

**Backend**
- Firebase (Firestore Database)

**배포**
- Vercel (or Firebase Hosting)
